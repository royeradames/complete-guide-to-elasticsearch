# Specifying the result format

## yaml

```JSON
# get yaml instead of json
GET recipe/_search?format=yaml
{
  "query": {
    "match": {
      "title": "pasta"
    }
  }
}
```

## pretty

`<index>/_search?pretty`

Useful in terminal. Get your response formatted.
