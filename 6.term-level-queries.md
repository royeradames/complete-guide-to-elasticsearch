# Searching for a term


Query for exact matching. So use it for searching no fields, dates or keyword fields.

```JSON
GET products/_search
{
  "query": {
    "term": {
      "is_active": true
    }
  }
}

# you need to do an object if you want to add options
GET products/_search
{
  "query": {
    "term": {
      "is_active": {
        "value": true
      }
    }
  }
}
```
